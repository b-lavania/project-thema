# Product dossier

Copy this folder to `products/<product-slug>/` when you own a product or major initiative.

## Files

| File | Purpose | PMOS bucket |
|------|---------|-------------|
| `brief.md` | Problem, users, hypothesis, scope | Strategy |
| `roadmap.md` | Now/next/later + cycle view (from `templates/02-roadmap.md`) | Strategy · Execution |
| `metrics.md` | KPI snapshots + feature→metric log | Analytics |
| `decisions.md` | Index linking to `../../decisions/` | Execution · Career |

## Quick start

```bash
cp -r PMOS/products/_template PMOS/products/<product-slug>
```

Fill `brief.md` first, then stand up `roadmap.md` and `metrics.md` in week one.
