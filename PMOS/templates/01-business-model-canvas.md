## Standard layout (9 blocks)

Populate left → right, top → bottom. Replicate this grid in Figma/Miro for workshops.

```
┌─────────────────┬─────────────────┬───────────────────────────────┐
│ Key Partners    │ Key Activities  │ Value Propositions            │
│                 ├─────────────────┤                               │
│                 │ Key Resources   │                               │
├─────────────────┴─────────────────┼───────────────┬───────────────┤
│ Channels                          │ Customer      │ Customer      │
│                                   │ Relationships │ Segments      │
├───────────────────────────────────┴───────────────┴───────────────┤
│ Cost Structure                    │ Revenue Streams               │
└───────────────────────────────────┴───────────────────────────────┘
```

## Canvas

| Block | Notes |
|-------|-------|
| **Key partners** | Who do we depend on? |
| **Key activities** | What must we do well? |
| **Key resources** | People, IP, data, brand |
| **Value propositions** | What job/pain/gain do we address? |
| **Customer relationships** | Self-serve, sales-led, community? |
| **Channels** | How do we reach and deliver? |
| **Customer segments** | Who are we building for? |
| **Cost structure** | Fixed vs. variable; biggest lines |
| **Revenue streams** | How does money flow in? |

## Fit check

- Where is the strongest product–market fit today?
- Which block is most fragile if we 2×?
- What would a competitor attack first?
