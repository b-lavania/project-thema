## Roles

| Role | Person | Responsibility |
|------|--------|----------------|
| **D — Driver** | | Runs process, gathers input, drafts recommendation |
| **A — Approver** | | Exactly one person who can approve or veto |
| **C — Contributors** | | Must be consulted; their input shapes the call |
| **I — Informed** | | Told once decided; no blocking vote |
