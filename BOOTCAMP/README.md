# PM Shock Cycle Bootcamp — Execution Kit

This directory contains everything needed to run the 30-day PM shock cycle. Each file is designed to be filled out, not just read.

## Files

| File | Purpose | Frequency |
|------|---------|-----------|
| `daily-scorecard.md` | Track quotas, punishments, and review | Every day |
| `weekly-audit.md` | Grade the week and plan the next | Every Sunday |
| `scoring-rubric.md` | Quantified points and penalties | Reference |
| `30-day-calendar.md` | Day-by-day assignments | Reference + daily |
| `drills/` | Templates for each drill | As assigned |

## How to start

1. Copy `daily-scorecard.md` to a new file named `scorecard-day-01.md`
2. Fill it out at the end of every day
3. Run the weekly audit every Sunday
4. Check the calendar each morning for the day's focus

**Not part of the main apply workflow.** Run this standalone app only during an active 30-day PM sprint (`streamlit run BOOTCAMP/app.py`). For job applications use the main app: Job Search → Generate → Outcomes (see `RES/docs/option_a_workflow.md`).
